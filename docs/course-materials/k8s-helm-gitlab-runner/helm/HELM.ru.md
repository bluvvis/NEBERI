# HELM

## Объяснение основ Helm:
- **Chart.yaml**: Определяет метаданные чарта, такие как имя, версия и описание.
- **values.yaml**: Содержит значения по умолчанию для конфигураций, используемых в шаблонах. Пользователи могут переопределить их при установке.
- **Шаблоны (Templates)**: YAML-файлы с плейсхолдерами и шаблонными директивами, которые рендерятся с использованием значений из `values.yaml` и пользовательских значений.
  - **Переменные**: Доступ осуществляется с помощью синтаксиса `{{ .Values.variableName }}`.
  - **Функции**: Вспомогательные функции, определённые в `_helpers.tpl`, подключаются с помощью `{{ include "functionName" . }}`.
  - **Условия и циклы**:
    - Используйте `{{- if }}`, `{{- else }}`, `{{- end }}` для условных конструкций.
    - Используйте `{{- range }}` для итерации по спискам или словарям.
- **_helpers.tpl**: Содержит переиспользуемые фрагменты шаблонов и вспомогательные функции для соблюдения принципа DRY (Don't Repeat Yourself — «Не повторяйся»).


## Установка

```shell
helm install quiner-fastapi-app fastapi_helm/
```

## Обновление

```shell
helm upgrade quiner-fastapi-app fastapi_helm/
```

## Удаление

```shell
helm uninstall quiner-fastapi-app fastapi_helm/
```

## Линтер (проверка синтаксиса)

```shell
helm lint fastapi_helm/
```
